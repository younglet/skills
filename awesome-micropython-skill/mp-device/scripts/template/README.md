## Quick Start

```bash
# 1. Check device
python ../awesome-micropython-skill/mp-device/scripts/device-info.py COM3

# 2. Upload
mpremote connect COM3 fs cp src/boot.py :boot.py
mpremote connect COM3 fs cp src/main.py :main.py

# 3. Run
mpremote connect COM3 run src/main.py

# 4. Monitor
python ../awesome-micropython-skill/mp-device/scripts/mem-monitor.py COM3 --watch 5
```
